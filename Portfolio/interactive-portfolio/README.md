# Portfolio

This is your Angular portfolio project.

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Install Angular CLI if you haven't:
   ```
   npm install -g @angular/cli
   ```

3. Serve the app:
   ```
   ng serve --open
   ```

## Build

Build for production:
```
ng build --prod
```
